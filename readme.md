## Run 开发 
```bash
    # 安装依赖
    npm install
    //or # 建议不要用cnpm  安装有各种诡异的bug 可以通过如下操作解决npm速度慢的问题
    npm install -g cnpm -registry=https://registry.npm.taobao.org


    # 本地开发 开启服务
    # develop
    npm run dev
```
浏览器访问  http://host:prot 

```

## Build 发布
```bash
    # 发布预发布环境
    npm run sit
    
    # 发布测试环境
    npm run test
    
    # 构建生成环境
    npm run build
```
